# はじめてgitに挑戦してみた！

gitをインストールして、ターミナルでメアドとユーザー名を登録するところまでは順調…。

で…でも…

『**ディレクトリ，ファイル作成の段階でもう既につまづいています！！**』

gitのコマンドの記述自体はシンプルで簡単なんだけど、  
実行してはエラー…実行してはエラーの繰り返し！！

GitHubでTIL（Today I Learned）リポジトリを作って、  
そこに学習記録を残していきたいけど、  
まずその土台作りが癖強すぎる！！

GitHubの「+Add file」からでもディレクトリもファイルも作れる…  
でもgitコマンドで作れないと後に響きそうだしなぁ…。
